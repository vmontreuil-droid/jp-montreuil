# JP Montreuil — Atelier Montreuil

Portfoliosite voor kunstenaar Jean-Pierre Montreuil. Migratie van [jp.montreuil.be](https://jp.montreuil.be) (WordPress / one.com) naar zelfgebouwde Next.js + Supabase site.

## Dev

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack

- Next.js 16 (App Router) + React 19
- Tailwind CSS 4
- Supabase (Postgres + Auth + Storage)
- Vercel hosting

Zie [AGENTS.md](./AGENTS.md) voor conventies en projectstructuur.
